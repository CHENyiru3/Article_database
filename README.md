# Article_database

Long-term organized collection for research and learning.

## Structure

- **Biological/** - Biology-focused: cell biology, immunology, developmental biology
- **CS_AI/** - CS/AI methodology: deep learning, transformers, foundation models
- **Comp_Bio/** - Computational methods: algorithms, pipelines, statistical methods
- **Others/** - Cross-disciplinary, reviews, or uncategorized

## Naming Convention

`Concise_Topic_Method_Year.md`

## Workflow

1. Read abstract from `/Notes/06_Mirror_pdf/`
2. Categorize and rename
3. Update `Log.md`
